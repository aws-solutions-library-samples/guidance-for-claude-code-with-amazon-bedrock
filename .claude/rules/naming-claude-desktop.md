description: When referring to Claude Desktop in code, docs, or comments
---
- Use "Claude Desktop" as the primary name (not "Cowork" or "CoWork")
- When a shorter reference is needed in code/config, use "cowork" (lowercase, one word) — this is the internal codename
- Never use "CoWork" (camelCase) — the correct form is "Cowork" (single capital)
- In user-facing docs and PR descriptions, prefer "Claude Desktop" over "Cowork"
- "Claude Desktop on 3P" or "Claude Desktop (third-party)" for the Bedrock-backed variant
